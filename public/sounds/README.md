# public/sounds/

This directory must contain a `rain.mp3` audio file for the "Rain" sound option in Deafening Silence.

## What to place here

A loopable rain recording. The file **must be licensed for redistribution** — do not use a random MP3 you found online.

## Recommended sources

Search [Freesound.org](https://freesound.org/) with:

- Query: `rain loop`
- Filter: License = **Creative Commons 0** (CC0 / public domain)

Direct search URL:
https://freesound.org/search/?q=rain+loop&filter=license%3A%22Creative+Commons+0%22

Alternatively, generate a synthetic rain loop with Audacity or similar tools and release it under CC0.

## File requirements

| Property      | Requirement                         |
|---------------|-------------------------------------|
| Filename      | `rain.mp3`                          |
| Format        | MP3 (or rename + update engine code)|
| Duration      | 10–60 seconds (will be looped)      |
| Sample rate   | 44100 Hz or 48000 Hz                |
| Licence       | CC0, CC-BY, or equivalent           |
